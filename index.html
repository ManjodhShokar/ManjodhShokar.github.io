MY GAME

My game, concluding the 2nd semester Programming in Java Module at City(20220. This project demonstrates my understanding of java, with things like arrays, inheritance, encapsulation, working with files, GUI etc. I created a game where the player character has to find 3 different keys around the map which will be used to open the lock at the end of the map. Opening the lock reveals a star which the player can walk into, which will take the player to the next level. Each level has unique challenges, from different enemy types, different environments and a bit of parkour. The player character has access to 2 different items, a pistol and a key, the pistol can be used to destroy enemies and the key is used to open the locks. Each level is vastly different from each other, they have different backgrounds, music, enemies, platforms and items.There are lives presented on the foreground which increases if you heal, or decrease if you take damage. The GUI allows the player to pause the game, go to the next level, adjust the volume of the game and to make save files. My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I created a forest scene in which details of each tree (e.g. position, trunk color and size, crown color and shape) are stored in parallel 2D arrays. Trees are arranged in a regular grid but their positions are jittered by small random offsets to create the impression of a real forest. Similarly, each tree trunk and crown has a individual color shade, obtained using random variations of brown and green. Finally, the forest changes over time: trees grow and their crowns become more pale to suggest seasonal change ( from spring to autumn).
